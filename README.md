# 🛒 বাজার দর (Bazar Dor)

**Real-time Market Price Intelligence Platform for Bangladesh 🇧🇩**

Built with **Next.js 15, TypeScript, Tailwind CSS 4, SQLite, Node.js, and modern data visualization tools**

---

## 🚀 Overview

**Bazar Dor** is a crowd-sourced, real-time market price tracking system designed for Bangladesh.

It enables users to:
- Track daily essential product prices
- Compare market rates across locations
- Detect fake or abnormal pricing
- Save money using smart planning tools
- Build a transparent, community-driven pricing ecosystem

---

## 🎯 Vision

To create a **trusted national price intelligence network** powered by real users, helping reduce price manipulation and improve market transparency in Bangladesh.

---

## 🧰 Tech Stack

### 🖥️ Frontend
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Ant Design (UI Components)
- Recharts (Data Visualization)
- Framer Motion (Animations)
- Lucide React (Icons)

### ⚙️ Backend
- Node.js (Express / API Routes)
- SQLite (better-sqlite3)
- REST API Architecture

### 🤖 AI Integration (Optional)
- Gemini API (AI-based suggestions & insights)
